🌎 My Country Explorer App

Explore the world from your browser! 🗺️ Search countries, view flags, capitals, population, time zones, and maps. Save your favorites and access them anytime with a beautiful, responsive design. ✨

🎯 Features

🔍 Search Countries: Quickly find any country by name.

🏳️ Country Details: Flag, capital, population, region, and more.

🗺️ Interactive Map: See country location with Leaflet.js.

💾 Save Favorites: Store your favorite countries in localStorage.

🖼️ Popup Modal: Scrollable popup to view saved countries, close easily.

⏰ Time Zones: Shows current local times for the selected country.

📱 Responsive Layout: Works on desktop and mobile; mobile uses CSS Grid.



⚡ Technologies Used

HTML5 & CSS3 – Page structure & styling

JavaScript (ES6+) – API calls, DOM manipulation

Leaflet.js – Interactive maps

REST Countries API – Fetch country data 🌐

LocalStorage – Save favorite countries locally

🚀 Setup & Installation
git clone https://github.com/yourusername/my-country-explorer.git
cd my-country-explorer

Open index.html in your browser, or run a local server:

npx live-server
🖱️ Usage

Type a country name and click Search 🔍

View country details: flag, capital, population, region

Save countries with the 🔖 button

Click Favorites 💖 to view saved countries in a modal popup

Close the popup with × or by clicking outside

Works beautifully on mobile devices 📱
